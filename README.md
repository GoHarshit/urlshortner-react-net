# Scalable URL Shortener

A full stack scalable URL shortener built using:

- React
- .NET 8 Web API
- PostgreSQL
- Redis
- JWT Authentication
- Docker
- Rate Limiting

# Features

- User Authentication
- Premium Plans
- Custom Aliases
- QR Code Generation
- Analytics
- Expiry Links
- Search
- Pagination
- URL Editing
- User Profiles
- Redis Caching
- Rate Limiting
- Dockerized Architecture

# Tech Stack

Frontend:
- React
- Axios
- CSS

Backend:
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- Redis

# Run Project

## Backend

dotnet run

## Frontend

npm install
npm run dev

# Docker

docker compose up --build
